HOW TO RUN TESTS:
=================
1. Ensure dev server is running: npm run dev
2. Run tests: npm run test:e2e
3. Run with UI: npm run test:e2e:ui
4. Run headed (see browser): npm run test:e2e:headed

Or directly:
npx playwright test __tests__/mobile-navigation.test.ts
